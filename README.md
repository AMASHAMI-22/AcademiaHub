# AcademiaHub - University Academic Website

## Project Description
AcademiaHub is a comprehensive university portal designed to provide students and faculty with easy access to academic resources, course information, and campus news. The website features a modern, responsive design to ensure a seamless user experience across all devices.

## Features
* **Home Page:** An overview of the university with latest announcements.
* **Responsive Design:** Optimized for desktops, tablets, and mobile phones.
* **Course Catalog:** Detailed information about available academic programs.
* **Clean UI:** Built with modern CSS techniques for a professional look.

## Technologies Used
* **HTML5:** For the website structure.
* **CSS3:** For styling and layout (Flexbox/Grid).
* **JavaScript:** For interactive elements.

## How to Run
Simply open the `index.html` file in any modern web browser to view the website.
